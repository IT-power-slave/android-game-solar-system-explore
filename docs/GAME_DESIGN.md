# Game Design: Math Heroes - Solar System Explorers

## 1. Overview
**Math Heroes: Solar System Explorers** is an educational experience designed for children and space enthusiasts. It combines high-quality 3D visualization with interactive learning and gamification.

## 2. Core Game Loop
1.  **Explore**: Navigating a 3D model of the Solar System.
2.  **Learn**: Clicking on celestial bodies to read detailed facts and "Wow-facts."
3.  **Quiz**: Testing knowledge through a time-pressured, multi-choice quiz.
4.  **Reward**: Earning XP (Points) and Badges (Achievements) to level up.

## 3. Game Modes
### A. Free Exploration
- Full 3D control (Orbit, Zoom).
- Information panels for each planet.
- Visual tracking of visited planets.

### B. Planet Lineup (Linear View)
- A guided, linear tour of the planets in order from the Sun.
- Automatic focus and information display.

### C. Planetary Quiz
- Randomly selected questions from a large pool.
- Scoring based on speed and accuracy.
- Streak bonuses.

## 4. Gamification Logic
### XP & Levels
- **Points**: 10 pts per correct answer, 5 pts per new planet visited.
- **Levels**:
    - *Student* (0 pts)
    - *Astronaut* (50 pts)
    - *Cosmonaut* (150 pts)
    - *Planet Explorer* (300 pts)
    - *Space Master* (500 pts)

### Badges
- *First Step*: Complete one quiz.
- *Perfection*: 10/10 in a quiz.
- *Explorer*: Visit all 8 planets.
- *Rocket*: Answer in < 5 seconds.
